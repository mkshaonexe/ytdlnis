🎯 HOW TO START THE APP:
========================

Just double-click this file: START.bat


✨ WHAT HAPPENS:
- Browser opens automatically
- You see the beautiful web interface
- Ready to download videos!


📥 HOW TO DOWNLOAD VIDEOS:
===========================

1. Paste a video URL (YouTube, Twitter, TikTok, etc.)
2. Click "Get Info" button
3. Choose Audio or Video
4. Click "Download" button
5. Done! Your video is downloading


📍 WHERE ARE VIDEOS SAVED:
===========================

Default location:
C:\Users\YourName\Downloads\YTDLnis

You can change this in Settings (gear icon)


🛑 TO STOP THE APP:
===================

Close the terminal window (press the X button)


🎨 FEATURES:
============

✅ Download from 1000+ websites
✅ Download audio or video
✅ Choose quality and format
✅ See download progress
✅ Download history
✅ Beautiful modern interface


📱 SUPPORTED SITES:
===================

YouTube, Twitter/X, Instagram, 
TikTok, Instagram, Facebook, 
Reddit, SoundCloud, Twitch, 
and 1000+ more!


💡 TIPS:
========

• For best quality: Select 1080p or 4K
• For music: Use Audio mode
• To save space: Lower the quality
• Multiple downloads: Queue as many as you want!


❓ TROUBLESHOOTING:
===================

Problem: App won't start
→ Make sure Node.js is installed from nodejs.org

Problem: Downloads slow
→ Check your internet connection

Problem: Port 3000 in use
→ Close other apps or restart computer
